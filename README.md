# Projekt Raspberry Camera Controller

Tento projekt slouží k ovládání Raspberry Pi kamery a řízení jejího pohybu pomocí tří krokových motorů.

## Plán projektu
- Ovládání kamery připojené k Raspberry Pi
- Pohyb kamery po třech osách pomocí krokových motorů
- Ovládání kamery a pohybu (např. přes webové rozhraní nebo terminál)
- Dokumentace zapojení hardware

## Struktura
- `otus.py` — hlavní skript v Pythonu
